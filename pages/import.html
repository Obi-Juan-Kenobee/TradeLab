Import Trades

Import from Excel

Upload your trades from an Excel file. We recommend using our template to make sure all trades import correctly. However, if you choose to import an excel sheet not in our format we recommend making sure these columns are present in your file.

  • Date: Trade date (MM/DD/YYYY)
  • Symbol: Trading symbol/ticker
  • Entry Price: Trade entry price
  • Exit Price: Trade exit price
  • Quantity: Number of shares/contracts

Note that for brokers with multi-row entries (one row for entry and one for exit, which should include most brokers), you can still import your trades, however the some data might be imported incorrectly. I am working hard on an algorithm that can parse excel sheets from most brokers.

Instructions

  1. Download our Excel template or ensure your file matches the required format
  2. Fill in your trade data following the template structure
  3. Upload your file using the drag & drop zone or file selector
  4. Review the preview of your data
  5. Click "Import Trades" to add the trades to your account

Required Columns

  • Symbol (e.g., AAPL, EURUSD)
  • Trade Type (Stock, Futures, Forex, Crypto, Options)
  • Entry Price
  • Exit Price
  • Quantity
  • Date
